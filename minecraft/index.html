我的世界服务器搭建笔记

Created2022-05-18|Updated2022-05-23
|Post View:

0. 前提

云服务器：腾讯云轻量服务器。2核4GB，带宽6Mbps

系统：Ubuntu Server 20.04 LTS 64bit

1. 配置java环境

自Java版1.18起需要Java 17启动游戏。

2. 安装MINECRAFT JAVA版服务器

我的世界服务器下载地址

创建minecraft文件夹并下载服务器


安装

修改eula.txt 把生成的eula.txt中的false改为true

3. 启动服务器

第一次启动的时间比较长，请耐心等待至100%

Xms：初始启动分配的内存

Xmx：最大分配的内存

nogui: 用于以基于文本的界面来显示，可减少内存使用。

关闭命令请使用stop，防止丢失进度。

4. 连接我的世界

服务端

服务端打开腾讯云控制台，在防火墙选项中添加规则：

类型：自定义

协议：TCP

端口：25565

策略：允许

端口号是server.properties中默认的，可以修改。

客户端

多人游戏，输入自己的服务器ip

5. 使用screen让服务器常开
